# Contributing

Branch-Workflow
- Feature branches: `feat/short-description`
- Fix branches: `fix/short-description`

Commits
- Use imperative, short messages, e.g. `feat: add time tracker`

Pull Requests
- Link issue (if vorhanden)
- Include short description of changes and testing steps
- Run `npx tsc -p tsconfig.json --noEmit` before opening a PR

Code Style
- Project uses Prettier and ESLint (configure locally)
