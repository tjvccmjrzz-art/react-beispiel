# React Beispiel

Kleine Demo-App mit React, TypeScript, Vite, `framer-motion`, `sonner` und IBAN-Input-Utilities.

Voraussetzungen
- Node.js 18+ empfohlen
- npm / pnpm / yarn

Schnellstart
```bash
npm install
npm run dev
```

Wichtige Befehle
- `npm run dev` — Development-Server (Vite)
- `npm run build` — Produktion bauen
- `npm run preview` — Lokale Preview des Builds
- `npx tsc -p tsconfig.json --noEmit` — TypeScript-Check

Kontribution
- Siehe `CONTRIBUTING.md` für Branch- & PR-Konventionen.

Remote-Repository & Push
- Um das lokale Repo auf GitHub zu pushen, erstelle ein Repo auf GitHub und dann:
```bash
git remote add origin git@github.com:YOUR_USER_OR_ORG/YOUR_REPO.git
git push -u origin main
```
